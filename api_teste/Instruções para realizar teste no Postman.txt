Para testar este código no Postman, siga os seguintes passos:

Inicie o servidor Flask executando o seguinte comando no terminal:

	python nome_do_arquivo.py


Abra o Postman e crie uma nova solicitação.

Na caixa de URL, digite o seguinte endereço:


	http://localhost:5000/recomendar


Selecione o método HTTP POST.

Na aba Body, selecione a opção raw e escolha JSON no menu suspenso.

Digite o seguinte JSON no corpo da solicitação:


{
    "feedback": "Eu amo pizza"
}
Lembre-se de que o texto em "feedback" pode ser substituído por qualquer outra entrada do usuário.

Clique em "Send" para enviar a solicitação.

Verifique a resposta da API na aba Body. A resposta deve conter uma mensagem de recomendação e uma lista de 10 restaurantes recomendados com base na entrada do usuário e na análise de sentimento.

Repita os passos 6-8 com diferentes entradas do usuário para testar diferentes cenários e resultados.
